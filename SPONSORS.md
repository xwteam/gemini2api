## 💖 支持项目

Gemini2API 是一个开源项目，主要由个人维护。如果这个项目对你有帮助，欢迎通过以下方式支持：

- ⭐ 给项目点个 Star
- 🔗 分享给有需要的朋友
- 🐛 提交 Issue 反馈 bug 或建议
- 🔧 提交 PR 贡献代码或文档

感谢所有在 Issues 里提交 bug 复现、日志、兼容性反馈和功能建议的用户。这些反馈直接推动了 Cookie 保活、多账号轮换、模型选择、多语言支持、Web 面板等核心能力的迭代。

## 🤝 参与贡献

1. Fork 本仓库
2. 创建分支 `git checkout -b feature/your-feature`
3. 提交代码 `git commit -m "feat: add something"`
4. 推送并创建 Pull Request

**贡献方向：**

- 🔑 Cookie 长期保活方案（目前约 2 小时失效，欢迎有经验的开发者贡献思路）
- 🌐 多语言翻译完善
- 📱 移动端适配优化
- 🧪 新模型兼容性测试
- 📖 文档改进
