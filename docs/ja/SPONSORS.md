# ☕ サポート & 貢献

役に立ちましたか？作者にコーヒーをおごるか、WeChatグループに参加してサポートを受けてください。QRコードはリポジトリのホームページ上部にあります。

---

## 💖 プロジェクトをサポート

Gemini2APIは主に個人によって維持されているオープンソースプロジェクトです。このプロジェクトがお役に立った場合は、以下の方法でサポートをご検討ください：

- ⭐ プロジェクトにスターを付ける
- 🔗 必要としている友人と共有する
- 🐛 バグや提案をIssueで報告する
- 🔧 コードやドキュメントをPRで貢献する

---

## 🙏 謝辞

Issuesでバグの再現、ログ、互換性フィードバック、機能提案を提出してくださったすべてのユーザーに感謝します。これらのフィードバックは、Cookie保活、マルチアカウントローテーション、モデル選択、多言語サポート、Webパネルなどのコア機能の反復を直接推進しました。

---

## 📢 コミュニティグループ

WeChatグループに参加して、使用方法のヘルプ、経験の共有、新機能の議論を行ってください。リポジトリのホームページ上部のQRコードをスキャンして参加できます。

---

## 🤝 貢献する

1. このリポジトリをフォークする
2. ブランチを作成する `git checkout -b feature/your-feature`
3. コードをコミットする `git commit -m "feat: add something"`
4. プッシュしてプルリクエストを作成する

**貢献分野：**

- 🔑 Cookie長期保活ソリューション（現在約2時間で失効、経験豊富な開発者のアイデア貢献を歓迎）
- 🌐 多言語翻訳の改善
- 📱 モバイル適応の最適化
- 🧪 新しいモデルの互換性テスト
- 📖 ドキュメントの改善
