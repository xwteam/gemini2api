## 💖 支持專案

Gemini2API 是一個開源專案，主要由個人維護。如果這個專案對你有幫助，歡迎透過以下方式支持：

- ⭐ 給專案點個 Star
- 🔗 分享給有需要的朋友
- 🐛 提交 Issue 回饋 bug 或建議
- 🔧 提交 PR 貢獻程式碼或文件

## 🤝 參與貢獻

1. Fork 本倉庫
2. 建立分支 `git checkout -b feature/your-feature`
3. 提交程式碼 `git commit -m "feat: add something"`
4. 推送並建立 Pull Request

**貢獻方向：**

- 🔑 Cookie 長期保活方案（目前約 2 小時失效，歡迎有經驗的開發者貢獻思路）
- 🌐 多語言翻譯完善
- 📱 行動端適配最佳化
- 🧪 新模型相容性測試
- 📖 文件改進
