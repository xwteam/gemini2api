## 💖 Support the Project

Gemini2API is an open-source project maintained primarily by an individual. If this project helps you, consider supporting it through:

- ⭐ Star the project
- 🔗 Share with friends who might need it
- 🐛 Submit Issues to report bugs or suggestions
- 🔧 Submit PRs to contribute code or documentation

## 🤝 Contributing

1. Fork this repository
2. Create a branch `git checkout -b feature/your-feature`
3. Commit code `git commit -m "feat: add something"`
4. Push and create a Pull Request

**Contribution Areas:**

- 🔑 Long-term Cookie keep-alive solutions (currently expires after ~2 hours, experienced developers welcome to contribute ideas)
- 🌐 Multi-language translation improvements
- 📱 Mobile adaptation optimization
- 🧪 New model compatibility testing
- 📖 Documentation improvements
